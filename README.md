## Installation
  Install the dependencies and devDependencies.
  
  ```sh
  npm init
  ```
   ```sh
  npm install inquirer
  ```   
   ```sh
  npm install mysql2
  ```
   ```sh
  npm install console-table
  ```
   ```sh
  npm install figlet
  ```
